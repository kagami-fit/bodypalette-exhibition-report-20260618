# ABOUT

## 一言で言うと

BODYPALETTEの展示会録音文字起こしから、健康経営に対する需要と代表の営業トークを分析した共有用HTMLレポートです。

FRACTAL WORKOUTの色味に合わせて、目次付きの全文レポートと、展示会前に短時間で見返せるクイック版を用意しています。

## 何ができるのか

- 展示会で実際に出た健康経営ニーズを確認できる
- 来場者タイプ別に刺さる訴求を確認できる
- 代表の営業トークの構造、強み、改善点を確認できる
- 次回展示会で誰でも話せる声かけ・30秒トーク・クロージングを確認できる
- 展示会直前に、5分で見返せるクイック版を確認できる
- 要約図解画像で、需要・営業トーク・データ活用・展示会再現の全体像を確認できる
- GitHub PagesでHTMLレポートとして共有できる

## 構成

- `index.html`
  - 共有用のHTMLレポート本体です。
  - GitHub Pagesではこのファイルが表示されます。
- `quick.html`
  - 展示会前にパッと見返すための要約ページです。
  - 30秒トーク、相手別訴求、クロージング、当日の注意点を短く整理しています。
- `assets/bodypalette-report-summary-diagram.png`
  - 全文レポート上部に配置している要約図解画像です。
- `BODYPALETTE展示会_需要営業分析レポート.md`
  - HTML化する前のMarkdown原稿です。
- `ABOUT.md`
  - この共有フォルダの説明です。

## 使い方

GitHub PagesのURLを開くと、HTMLレポートをブラウザで閲覧できます。展示会前に短時間で確認したい場合は、ページ上部の「展示会前クイック版」ボタンから `quick.html` を開きます。

ローカルで確認する場合は、`index.html` または `quick.html` をブラウザで開きます。
