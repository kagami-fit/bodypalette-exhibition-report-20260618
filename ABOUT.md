# ABOUT

## 一言で言うと

BODYPALETTEの展示会録音文字起こしから、健康経営に対する需要と代表の営業トークを分析した共有用HTMLレポートです。

FRACTAL WORKOUTの色味に合わせて、目次付きで読みやすく整えています。

## 何ができるのか

- 展示会で実際に出た健康経営ニーズを確認できる
- 来場者タイプ別に刺さる訴求を確認できる
- 代表の営業トークの構造、強み、改善点を確認できる
- 次回展示会で誰でも話せる声かけ・30秒トーク・クロージングを確認できる
- GitHub PagesでHTMLレポートとして共有できる

## 構成

- `index.html`
  - 共有用のHTMLレポート本体です。
  - GitHub Pagesではこのファイルが表示されます。
- `BODYPALETTE展示会_需要営業分析レポート.md`
  - HTML化する前のMarkdown原稿です。
- `ABOUT.md`
  - この共有フォルダの説明です。

## 使い方

GitHub PagesのURLを開くと、HTMLレポートをブラウザで閲覧できます。

ローカルで確認する場合は、`index.html` をブラウザで開きます。
