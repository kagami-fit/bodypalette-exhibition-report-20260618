## 1. 全体結論

今回の展示会で実際に出ていた需要は、単に「健康経営をやりたい」ではありません。来場者の本音は、次の7つに集約されます。

1. 健康経営優良法人の申請・更新の工数を減らしたい
2. 認定を取っただけで終わらせず、社員の健康課題を本当に改善したい
3. 健診・ストレスチェック・アンケートのデータを取っているが、次に何をすべきかわからない
4. 施策をやっても、効果やROIを説明できない
5. 高齢者、女性、更年期、転倒防止、夜勤、現場仕事、立ち仕事など、自社特有の課題に合う施策が欲しい
6. 特定保健指導該当者、生活習慣病、健診後フォローなど、より医療寄りの改善プログラムが欲しい
7. 保険会社、健診ベンダー、アプリ事業者、医療機関、地域支援者が、自社顧客に紹介できる具体的な健康経営ソリューションを探している

BODYPALETTEが強く刺さっているのは、「認定サポート」そのものではなく、次の組み合わせです。

> 健康課題を見える化し、優先順位をつけ、医療連携・運動指導の専門性をもとに、具体施策まで実行し、結果を振り返れること。

代表の営業トークは、この価値を相手に合わせて言い換える力が非常に強いです。一方で、展示会で誰でも再現するには、情報量が多すぎるため、話法を「相手タイプ別」に整理する必要があります。

## 2. 需要の全洗い出し

### 2.1 認定・申請・更新まわりの需要

| 需要 | 来場者の状態 | BODYPALETTEの刺さる訴求 |
|---|---|---|
| 健康経営優良法人の申請を初めてやりたい | 何から始めればよいかわからない | 申請要件に合わせたeラーニング、アンケート、施策まで一気通貫で支援 |
| 毎年の更新作業が大変 | 既に認定取得済みだが、担当者の工数が重い | 更新工数を大幅削減、必要項目の多くをツールとコンテンツでカバー |
| 担当者が変わってノウハウが残らない | 総務・人事担当者が新任 | ユーザー管理、eラーニング履歴、アンケート履歴で属人化を減らす |
| 申請要件が毎年変わることへの不安 | 7月の新要件発表後に慌てる | 要件変更に合わせてコンテンツ更新、申請期間中の駆け込み支援 |
| ホワイト500や大規模法人の上位認定を狙いたい | 既に基本認定は取得済み | アウトカム指標、数値管理、施策前後比較を支援 |
| 健康経営の担当者が少ない | 2-3名体制、他業務兼務 | ツール、セミナー、コンサル的伴走で実務を外部化 |
| 社内稟議・経営層説明が難しい | 「なぜ健康経営に予算を使うのか」を説明できない | 採用、離職、生産性、医療費、プレゼンティーズムに接続して説明 |

録音内では、既取得企業だけでなく、「これから取る」「上がOKしない」「担当に資料を持ってこいと言われた」というライトな接点も複数ありました。初回接点では、認定要件の細かい説明よりも「今のステージ確認」と「次に必要な一手」を短く出す方が効果的です。

### 2.2 データ活用・見える化まわりの需要

| 需要 | 来場者の状態 | BODYPALETTEの刺さる訴求 |
|---|---|---|
| 健診データはあるが、活用できていない | 健診代行・健診管理ツールは導入または検討中 | 健診後のネクストアクションをBODYPALETTEが担う |
| ストレスチェックをやって終わり | 高ストレス者に「産業医に行ってね」で終わっている | 月次アンケートで変化を追い、必要なら産業医・外部専門家につなぐ |
| 健康課題が何かわからない | 福利厚生はあるが使われていない | 18項目アンケートでワースト課題を可視化 |
| 部署別・年代別・性別で課題を見たい | 現場、営業、事務、シフト勤務などが混在 | 部署・属性別に集計して施策を変える |
| 施策が効いたか測りたい | セミナーや福利厚生を実施しても成果が不明 | 前後比較、月次推移、プレゼンティーズム指標で見る |
| 食事、運動、睡眠、メンタルを横断して見たい | 単一テーマのサービスでは不足 | 総合アンケートと専門家ネットワークで課題別に対応 |
| ROIを出したい | 経営層に説明できない | 課題に対して費用を使い、改善幅を報告する設計 |

重要なのは、来場者の多くが「データを取りたい」ではなく「データを見た後に何をすればよいか知りたい」と話している点です。ここはBODYPALETTEの差別化の中心です。

## 3. 来場者タイプ別の刺さる訴求

### タイプA: 健康経営をすでに取得している人事・総務

主な悩み:

- 毎年の更新が大変
- 施策がマンネリ化している
- 何をやればよいかわからない
- 効果測定ができない
- 経営層に成果を説明できない

刺さる一言:

> 認定更新の工数を下げながら、社員の健康課題をデータで見える化して、施策の効果まで追えるサービスです。

聞くべき質問:

- 今年も更新予定ですか？
- 更新作業は何名体制でされていますか？
- 認定取得後、施策が効いているかは見えていますか？
- 毎年「今年は何をやるか」で悩んでいませんか？

提案:

- 月次アンケート
- eラーニング
- 課題別セミナー
- 申請要件に紐づく施策証跡
- ROI・前後比較レポート

### タイプB: これから健康経営に取り組む企業

主な悩み:

- 何から始めるべきかわからない
- 健診データが紙やExcelで管理されている
- 役員承認が降りていない
- 健康経営の必要性を説明できない

刺さる一言:

> 初めての申請に必要な土台づくりから、社員に効く施策づくりまでまとめて支援できます。

聞くべき質問:

- 今は申請前ですか、それとも情報収集中ですか？
- 従業員数は何名くらいですか？
- 健診データやストレスチェックはどう管理されていますか？
- 役員のOKが出ない理由は、費用ですか、効果が見えないことですか？

提案:

- 無料アンケート
- 健康課題の棚卸し
- 初回セミナー
- 申請要件の不足チェック
- 役員説明用の健康経営ストーリー

刺さる一言:

> 認定をゴールにせず、社員の実際の不調を見つけて、改善まで伴走する健康経営です。

聞くべき質問:

- 認定は重視していますか、それとも社員改善を優先していますか？

- 18項目アンケート
- 課題ワースト5の可視化
- 課題に応じたセミナー
- 使われない福利厚生の見直し

### タイプD: 現場・工場・警備・施設管理・立ち仕事系

## 4. 代表の営業トーク詳細分析

### 4.1 代表トークの基本構造

代表の会話は、おおむね次の流れで進んでいます。

1. 短い声かけ
   - 「健康経営のサポートをしています」
   - 「従業員の健康状態を見える化しています」
   - 「健康経営の工数を大幅削減できます」
   - 「健康経営のROIまで見える化できます」

2. 相手のステージ確認
   - 健康経営はやっているか
   - 認定は取得済みか、これからか
   - 大規模法人か中小規模法人か
   - 担当者本人か、資料収集役か
   - 既存ベンダーはいるか

3. 課題のヒアリング
   - 何に困っているか
   - 健診データやストレスチェックは活用できているか
   - 社員の不調は何か
   - 離職、夜勤、現場、高齢化など経営課題はあるか

4. 相手の文脈に合わせたリフレーミング
   - 人事には「工数削減」「施策効果」
   - 経営層には「採用・離職・生産性」
   - 現場には「足腰・転倒・労災」
   - 健診ベンダーには「データ後のネクストアクション」
   - 保険会社には「顧客企業への紹介商材」

5. BODYPALETTEの強み提示
   - アンケートで健康課題を見える化
   - ワースト課題に対して施策を打つ
   - 医師から処方箋を受けて運動療法を提供している専門性
   - 運動・食事・メンタル・女性・高齢者など幅広く対応
   - セミナー、eラーニング、オンサイト、オンライン、オンデマンド

6. 具体提案
   - まず無料アンケート
   - 3ヶ月プログラム
   - 単発セミナー
   - 東京・大阪など拠点別実施
   - 役員説明用の資料化

7. CTA
   - QRコード取得
   - 後日打ち合わせ
   - 担当者紹介
   - 資料送付
   - 体験セミナー

### 4.2 代表トークの強み

#### 強み1: 相手の業種・状況に即座に合わせる

代表は、相手の業種や職種を聞いた瞬間に、話をかなり細かく変えています。

- ホテル・シフト勤務には、睡眠、夜勤、離職
- 警備・施設管理には、足腰、転倒、労災
- 健診ベンダーには、健診後フォロー、特保、API連携
- 保険会社・銀行には、顧客企業への紹介、手数料・値引き
- 人事・総務には、工数削減、更新、施策効果
- 経営企画には、採用、離職、生産性

これは非常に強いです。ただし属人的です。今後は、スタッフが再現できるように「相手タイプ別トークカード」に落とすべきです。

#### 強み2: 「データを取る」ではなく「データの後」を売っている

代表は何度も、健診データ・ストレスチェック・アンケートは取って終わりになりがちだと話しています。

BODYPALETTEの強みは次です。

> データ取得後に、課題の優先順位をつけ、具体施策までやる。

このポジションは展示会でかなり刺さっています。健康管理ツール、健診代行、ストレスチェックベンダーが多い場では、差別化しやすいです。

#### 強み3: 医療連携・運動療法の信頼性を出せる

代表は、原宿の施設、医師の処方箋、医療費控除、生活習慣病、整形外科、婦人科、精神科、循環器系などの経験を話しています。

これにより、単なるフィットネス企業ではなく、

> 医療と企業健康経営の間をつなぐ会社

として見えます。

展示会での訴求は、次の一文にするとわかりやすいです。

> 私たちは、医師の処方を運動指導に翻訳してきた会社なので、企業の健康課題も「何をすればよいか」まで具体化できます。

#### 強み4: 具体例が多い

録音内で出ていた具体例:

- 従業員2000名の工場で転倒防止
- 警備会社1000名規模、現場80%以上
- 女性の痩せ・更年期プログラム
- 3ヶ月で数値改善を狙う特保向けプログラム
- 立ち仕事の足底筋膜炎・膝痛・股関節ケア
- 夜勤・シフト勤務の睡眠・食事課題
- 東京・大阪拠点別の実施
- 健診ベンダーとの連携

これらは事例カード化すべきです。代表以外が話すときも、事例があるだけで信頼度が上がります。

#### 強み5: 経営課題への接続がうまい

特に、事業拡大中の企業に対して「採用」「離職」「長く働ける環境」に接続していた点が強いです。

健康経営を福利厚生ではなく、経営課題の解決として語っています。

使える言い回し:

> 事業拡大には人が必要です。採用だけでなく、辞めない環境づくりも必要です。そのための健康経営として設計しましょう。

### 4.3 代表トークの改善点

#### 改善点1: 情報量が多く、初回接点では重くなることがある

代表は知識量が多く、相手が関心を持つと一気に深く話せます。一方で、展示会では相手の理解速度を超える可能性があります。

改善:

- 最初の30秒は必ず短くする
- 相手が食いついたテーマだけ深掘る
- 「全部できます」ではなく「まず御社ならここです」と絞る

#### 改善点2: 声かけが日中で何度も変わっている

録音中で声かけは何度も試行錯誤されています。

使われていた主な声かけ:

- 健康経営のサポートをしています
- 従業員の健康状態を見える化しています
- 健康経営の工数を大幅削減できます
- 健康経営のROIを見える化できます
- 会社の健康課題を一緒に解決します
- 認定だけでなくリアルな課題解決まで行います

反応がよさそうだったのは、単なる「健康経営サポート」よりも、次の2つです。

- 「工数削減」
- 「ROI・費用対効果」

ただし、健康経営未経験者には「工数削減」は刺さりにくいので、ステージ別に変えるべきです。

#### 改善点3: 料金説明が複雑に聞こえる

録音では、月額、ID単価、基本料金、オンサイト、単発、契約企業向け価格、展示会価格、足代などが会話内で混ざります。

改善:

- 展示会では3パッケージだけ出す
- 詳細見積もりは後日
- 「まず単発」「3ヶ月トライアル」「年間伴走」の3段階にする

推奨パッケージ:

1. 単発セミナー
   - オンライン
   - オンサイト
   - 課題別テーマ

2. 3ヶ月課題改善トライアル
   - 初回アンケート
   - 月1セミナー
   - 前後比較レポート

3. 年間伴走プラン
   - 健康経営申請・更新支援
   - 月次モニタリング
   - eラーニング
   - オープンセッション
   - カスタム施策

#### 改善点4: 「何でもできます」は便利だが、印象がぼやける

代表は本当に広く対応できるため「何でもできます」と言えます。ただ、来場者には選びづらくなります。

改善:

> 何でもできますが、御社の場合はまず「〇〇」からです。

この形にする。

例:

- 現場企業: 「まず転倒防止と足腰ケアです」
- 既取得企業: 「まず施策効果の見える化です」
- 初取得企業: 「まず申請要件と健康課題の棚卸しです」
- 健診ベンダー: 「まず健診後フォローです」

#### 改善点5: 制度・医療・効果の表現は慎重に整える

医療費控除、処方箋、助成金、認定要件、数値改善などは非常に強い一方で、誤解があるとリスクがあります。

改善:

- 展示会用の「言ってよい表現集」を作る
- 「保証」ではなく「改善を目指す」「支援する」「設計する」と言う
- 最新制度は「確認のうえご案内します」とする
- 医療行為ではなく「運動指導」「生活習慣改善支援」と明確化する

## 5. 次回展示会用トーク設計

### 5.1 最初の声かけテンプレート

#### 汎用

> 健康経営の認定更新と、社員の健康課題の見える化・改善まで支援しています。

#### 既に健康経営をやっていそうな人向け

> 健康経営、毎年の更新や施策の効果測定で困っていませんか？

#### 人事・総務向け

> 健康経営の担当者工数を減らしながら、社員の不調まで見える化できます。

#### 経営企画・経営層向け

> 健康経営を、採用・離職・生産性の改善に接続する支援をしています。

#### 現場系・施設系向け

> 現場の足腰・転倒・疲労など、リアルな健康課題に合わせた施策を作れます。

#### 健診・健康管理システム事業者向け

> 健診データやストレスチェックの後の、具体的な改善施策を一緒に作れます。

#### 認定に懐疑的な人向け

> 認定のためだけではなく、社員の実際の不調を見つけて改善する健康経営です。

### 5.2 30秒トーク

> BODYPALETTEは、健康経営の認定サポートだけでなく、社員の健康状態をアンケートで見える化して、課題に合わせたセミナーや運動施策まで実行するサービスです。健診やストレスチェックは取って終わりになりがちですが、当社は「次に何をするか」まで伴走します。肩こり・腰痛・眼精疲労・睡眠・運動不足・特定保健指導・高齢者の転倒防止など、御社の課題に合わせて設計できます。

### 5.3 2分トーク

> 健康経営は、認定を取ること自体よりも、その後に社員の健康課題をどう改善するかが重要です。多くの企業では、健診データやストレスチェックは取っているものの、そこから何をすべきかが見えていません。  
>  
> BODYPALETTEでは、まず18項目のアンケートで体調、睡眠、食事、運動習慣、ストレス、プレゼンティーズムなどを見える化します。その結果をもとに、例えば肩こり・腰痛が多ければオフィスでできるストレッチ、夜勤者が多ければ睡眠・食事、シニア現場職が多ければ転倒防止、特保該当者が多ければ3ヶ月改善プログラムというように、施策まで実行します。  
>  
> もともと当社は医師と連携して運動療法を提供している背景があるので、単なる福利厚生ではなく、健康課題に合わせた専門的な施策設計ができます。

### 5.4 深掘り質問リスト

最初に聞く:

- 健康経営はすでに取り組まれていますか？
- 認定は取得済みですか、それともこれからですか？
- ご担当は人事・総務の方ですか？
- 従業員数は何名くらいですか？

課題を聞く:

- 今、一番困っているのは申請工数ですか、施策内容ですか、効果測定ですか？
- 健診データやストレスチェックは活用できていますか？
- 社員の不調は何が多そうですか？
- 現場職、デスクワーク、シフト勤務のどれが多いですか？
- 離職、休職、体力、睡眠、メンタルの課題はありますか？

提案につなげる:

- まず無料アンケートで課題を見える化してみませんか？
- 御社なら最初は〇〇のテーマがよさそうです。
- 単発で試すか、3ヶ月で前後比較する形が合いそうです。
- 担当者様向けに、後日15分だけオンラインでご説明できます。

### 5.5 クロージングの型

#### QR取得

> 具体的な資料と、無料アンケートのご案内をお送りしますので、こちらだけ読み取っていただいてもよろしいですか？

#### 後日商談

> 今日のお話だと、御社は〇〇が課題になりそうです。そこに絞って15分で具体案をお持ちしたいので、後日お時間いただけますか？

#### 担当者紹介

> もし本来のご担当者様が別にいらっしゃれば、その方宛てに「御社なら〇〇の話が合いそうです」と短くお送りします。

#### 体験提案

> 社員の方に一度体験していただくとかなり伝わりやすいので、単発のオンラインまたはオンサイトで試す形もできます。

## 7. 展示会運用の分析

### 7.1 ブースで反応が取れていた要素

反応が取れていたもの:

- アバター接客の物珍しさ
- 「健康経営の工数削減」
- 「ROI・費用対効果」
- 「健康状態の見える化」
- 「具体的な課題解決まで伴走」
- 「単発セミナーも可能」
- 代表本人の専門性と具体例
- 医療連携・運動療法処方箋の話

一方で、アバターは足止め・話題化には役立つものの、商談化は人の介入が必要です。アバターで興味を引き、人が質問で選別する設計がよさそうです。

### 7.2 声かけで試されていた軸

| 声かけ | 向いている相手 | コメント |
---|---|---|
| 健康経営のサポートをしています | 汎用 | 弱くはないが差別化は薄い |
| 従業員の健康状態を見える化しています | 人事・総務 | 説明につなげやすい |
| 健康経営の工数を大幅削減できます | 既取得企業 | 足を止めやすい |
| 健康経営のROIを見える化できます | 経営企画・大企業 | 刺さると深い |
| 会社の健康課題を一緒に解決します | 認定懐疑派・本質派 | BODYPALETTEらしい |
| 認定だけでなくリアルな課題解決まで | 既取得・マンネリ企業 | 差別化が強い |

推奨:

- 午前・初回接点は「工数削減」「見える化」
- 人事・総務には「更新が楽になる」
- 経営層には「ROI」
- 現場系には「足腰・転倒・疲労」
- 健診ベンダーには「健診後の次の一手」

### 7.3 来場者の見分け方

録音内で言及されていた有望層:

- 人事
- 総務
- 経営企画
- 経営者
- DX・IT
- 健康経営担当者
- 健診・健康管理システム関係者
- 保険・金融・地域支援系
- 現場管理・労働衛生管理者

会話の入り口:

- 名札の部署を見て声かけを変える
- 「健康経営やられていますか？」だけでなく「担当者工数」「ROI」「現場の足腰」など相手に合わせる
- 資料収集役には、担当者に渡しやすい一言を添える

### 7.4 展示会後フォローの優先順位

#### Sランク: すぐ商談化

条件:

- 具体課題を話している
- 名刺交換・QR取得済み
- 後日打ち合わせの流れがある
- 予算・日程・拠点・人数の話が出た

該当例:

- 既取得で経営層説得・離職・睡眠・シフト課題がある企業
- 現場の足底筋膜炎・労災・転倒防止課題が具体的な企業
- 健診システム乗り換え検討中で特保連携に関心がある企業
- 東京・大阪で運動スタッフ派遣を検討した企業

フォロー内容:

- 当日話した課題を1行で振り返る
- 「御社ならまず〇〇」と提案
- 15-30分のオンライン商談打診

#### Aランク: 育成すべき

条件:

- 健康経営に関心あり
- 課題は曖昧
- 担当者ではない
- 資料収集段階

フォロー内容:

- 無料アンケート案内
- セミナー一覧送付
- 担当者紹介依頼

#### Bランク: パートナー候補

条件:

- 健診・ストレスチェック・アプリ・デバイス・保険・医療機関
- 自社顧客への紹介や連携余地あり

フォロー内容:

- 「データ取得後の施策実行を担える」資料
- 共同提案パターン
- API/紹介/共同PRの選択肢

#### Cランク: 情報収集

条件:

- 担当ではない
- チラシだけ
- 興味が薄い

フォロー内容:

## 8. 今後の展示会で誰でも話せるようにするための型

### 8.1 まず覚えるべき一文

> BODYPALETTEは、健康経営の認定取得・更新だけでなく、社員の健康課題を見える化し、具体的な施策まで実行する伴走型サービスです。

### 8.2 次に覚えるべき3つの強み

1. データで課題を見える化する
2. 課題に合わせてセミナー・運動施策まで実行する
3. 医療連携した運動指導の専門性がある

### 8.3 相手別の一言変換

| 相手 | 一言 |
---|---|
| 健康経営担当 | 更新工数を下げながら、施策効果まで見える化できます |
| 初めての企業 | 申請に必要な土台づくりから支援できます |
| 経営層 | 採用・離職・生産性に効く健康経営にできます |
| 現場系企業 | 足腰・転倒・疲労など現場の不調に合わせた施策ができます |
| 健診ベンダー | 健診後の具体的な改善施策を担えます |
| 福利厚生担当 | 使われる福利厚生にするために、社員の課題から設計できます |
| 認定懐疑派 | 認定のためでなく、社員の実課題を改善する設計です |

### 8.4 誰でも使える会話フロー

1. 声かけ
   - 「健康経営はもう取り組まれていますか？」

2. ステージ確認
   - 「認定は取得済みですか？これからですか？」

3. 困りごと確認
   - 「工数、施策内容、効果測定だと、どこが一番大変ですか？」

4. 課題仮説
   - 「御社だと、まず〇〇がテーマになりそうですね」

5. サービス説明
   - 「アンケートで課題を見える化して、課題に合わせたセミナーまで行います」

6. 具体例
   - 「例えば現場なら転倒防止、デスクワークなら肩こり・眼精疲労、シフト勤務なら睡眠・食事です」

7. CTA
   - 「無料診断のご案内を送るので、QRだけお願いできますか？」

### 8.5 代表っぽさを再現するための話し方

代表の話し方の特徴:

- 相手の言葉を拾う
- すぐに「それなら〇〇ですね」と具体化する
- 健康施策を経営課題に接続する
- 「取って終わりじゃない」と繰り返す
- 専門性を出しつつ、相手の業務負担を下げる話に戻す
- 最後は必ず次アクションを取る

スタッフが真似すべき言い回し:

- 「まず課題を見える化しましょう」
- 「御社の場合は〇〇からがよさそうです」
- 「データを取って終わりではなく、次に何をするかまで支援します」
- 「認定だけでなく、社員にちゃんと効く施策にしましょう」
- 「まずは単発でも、無料アンケートでも始められます」

- FAQを作る
- 禁止表現・推奨表現を作る
- 料金表を簡略化する
- トークカードを作る
- 事例で話す
